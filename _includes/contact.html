Contact

 

The best way to get help and advice is to post a message on the forum. The FAQ and the Infer.NET team blog also provide useful information.

There are also many tutorials and examples which you can often use as a starting off point. The installation includes a Visual Studio solution which allows you to run and edit these examples. If you don't already have a copy of Visual Studio, you can download the Visual Studio Community product.

If you prefer to get in contact directly, you can e-mail infersup@microsoft.com.
